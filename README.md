# Análisis Determinista Multicapa – Sapolsky

Aplicación en Streamlit para analizar personalidad y conducta desde un marco
determinista causal inspirado en Robert Sapolsky.

## Features
- Cuestionario 100% Likert
- Modelo de 7 capas causales
- Análisis automático de puntos críticos de intervención
- Sin libre albedrío, sin juicios morales

## Deploy
1. Subir a GitHub
2. Conectar repo en Streamlit Cloud
3. Listo
